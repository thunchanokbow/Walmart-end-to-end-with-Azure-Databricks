# Walmart-end-to-end-with-Azure-Databricks
This project integrates data from websites using Azure Data Factory. The data is extracted in batches and transformed using Spark in Azure Databricks. The transformed data is stored in Synapse Analytics in columnar format. Finally, a dashboard is created using Looker Studio or Power BI.
